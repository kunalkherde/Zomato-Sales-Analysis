# 🍽️ Zomato Restaurant Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using a dataset of 50,000+ restaurant records. The goal is to analyze restaurant performance, customer engagement, pricing strategies, and online delivery behavior.

The dashboard helps in understanding key business insights such as top-performing restaurants, popular cuisines, cost distribution, and location-based trends.

---

## 🎯 Objectives

* Analyze restaurant distribution across locations
* Identify top-performing restaurants based on ratings and votes
* Understand customer engagement and popularity trends
* Explore pricing patterns and cost behavior
* Evaluate online ordering and table booking availability

---

## 📊 Dashboard Features

* 6 interactive report pages
* KPI cards for quick insights
* Dynamic filters and slicers
* Visualizations including bar charts, column charts, donut charts, maps, and line charts

---

## 📁 Key Insights

* High-rated restaurants tend to receive more customer votes
* Certain locations dominate in terms of restaurant density
* Mid-range pricing categories have the highest number of restaurants
* Online ordering is more common in popular restaurant types

---

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation (Power Query)

---

## 📌 Dataset

* Zomato restaurant dataset (50k+ records)
* Includes details like restaurant name, location, rating, votes, cost, cuisine, and service availability

---

## 🚀 Conclusion

This dashboard provides a comprehensive view of the food delivery ecosystem and helps in making data-driven decisions related to pricing, customer engagement, and service offerings.
